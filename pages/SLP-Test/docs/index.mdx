# Perzeptron

## Überblick
Dieses Projekt begleitet dich beim Einstieg in Neuronale Netze, indem ein einfaches neuronales Netz aufgebaut wird.
Dabei lernst du die Grundlage der Neuronalen Netze kennen und wie diese in Python umgesetzt werden können.

## Ziel
Das Ziel des Projekts ist es ein neuronales Netz, das aus einer Schicht besteht, von Grundauf in Python zu implementieren.
